# Codogotchi execution creed (June 2026)

Date: 2026-05-27  
Purpose: daily decision gate to ship meaningful value fast

## Non-negotiable stance

- The idea is not unique anymore.
- Someone else can ship a "Codogotchi-like" product.
- Winning comes from speed-to-value, not conceptual originality.

## Mission to June 30

Ship a useful, defensible macOS product to market fast, with clear user value and launch discipline.

## Core promise

Codogotchi is not "just a pet wrapper."  
Codogotchi is a **cross-agent workflow companion** with meaningful state semantics and progression potential.

## Daily decision filter (must pass all)

1. Does this increase App Store launch probability?
2. Does this improve user-visible value this week?
3. Does this strengthen our moat (workflow truth + progression)?
4. Can this ship in a small slice quickly?

If a task fails this filter, defer it.

## Must-win priorities (in order)

1. App Store readiness and review safety.
2. Product trust and stability.
3. Clear differentiation in UX (state truth, attention clarity, progression signal).
4. Onboarding speed and compatibility.
5. Nice-to-have polish.

## What we ship, not what we admire

Ship increments that users can feel:

- better explanation of what the pet is doing,
- less "stuck/ambiguous" behavior,
- visible progression tied to real work,
- faster path from install to "this is useful."

## Hard no list

- Architecture rewrites without launch impact.
- Perfection passes that delay release.
- New feature branches that do not move launch or value metrics.
- Competing as a generic pet marketplace.

## Operating rules

- Small slices over grand rewrites.
- Freeze scope aggressively near launch windows.
- Resolve unknown-high-risk items early.
- Ship, measure, iterate.

## Weekly scorecard (minimum bar)

Every week must deliver:

- one launch-risk reduction,
- one user-visible product improvement,
- one moat-strengthening improvement.

If a week delivers none, reset plan immediately.

## Messaging guardrail

Never lead with "cute pet."  
Always lead with: **cross-agent workflow signal + progression value**.

## Execution mantra

Hunker down.  
Deliver.  
Ship fast.  
Do not let perfection be the enemy of good.
