**Finding 1** ŌĆö `apps/menubar/Sources/DemoCycleDriver.swift`, lines 109ŌĆō118

**Recommended fix:** Add an `else` NSLog when the `case .success` arm fails:
```swift

       lastEmittedState != entry.state {
        log.recordTransition(snapshot: snapshot, previousState: lastEmittedState ?? entry.state)

        NSLog("DemoCycleDriver: transition log skipped ŌĆö fixture parse failed (\(err))")

```
